My name is Alexander, though I prefer to go by Alex. I'm a full stack web developer, graduating from Thinkful's Engineering Flex program.

Back-end APIs were designed using RESTful API building practices, with robust validation middleware, being mindful of both the time- and space-complexities of each function.

Intelligent route handling responding with appropriate HTTP status codes, JSON formatted data, and robust error handling for invalid requests or methods.
