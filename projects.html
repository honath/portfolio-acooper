Decoder Ring

This functions primarily as a placeholder for a future, larger project - however, the Decoder Ring project uses Node.js to take in a string and either encode or decode the message, based on the parameters provided, and utilizes the following ciphers:

  • Caesar Shift
  • Polybius Square
  • Substitution Cipher
