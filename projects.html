Decoder Ring

This functions primarily as a placeholder for a future, larger project - however, the Decoder Ring project uses Node.js to take in a string and either encode or decode the message, based on the parameters provided, and utilizes the following ciphers:

  • Caesar Shift
  • Polybius Square
  • Substitution Cipher

The GitHub repository can be accessed here: project-decoder-ring

The Pomodoro Timer project makes use of Bootstrap and React to create a dynamic timer making use of the Pomodoro technique. The Pomodoro technique separates work into intervals, separated by a small break. There are upper and lower bounds to the focus and break timers, however both are adjustable via buttons.

The GitHub repository can be accessed here: Project_Pomodoro_Timer
